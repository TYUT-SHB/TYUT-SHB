# 清华的污点不该由太原理工背锅，但太原理工的资源决不能任由收割

作为一个土生土长的山西人，曾对每一位援晋的高端人才寄予厚望，期待他们能在这片黄土地上重振太原理工雄风。然而，随着孙宏斌任职太原理工大学副校长以来的种种迹象浮出水面，不得不审视一个残酷的现实：有些人的到来，不是为了耕耘，而是为了收割。长期以来，关于孙宏斌在十年前控制美国电网、国家奖争议的种种污点，常被舆论张冠李戴到太原理工头上。那些陈年旧账是清华的耻辱，与太原理工无关。但在太原的真相是，孙宏斌自2020年入职后，将太原理工当作了申报院士头衔的提款机，到头来顶着清华大学的光环评上了中国科学院院士。

## 一、 巧取豪夺：将学校资产化作个人院士申报垫脚石

　　自2020年起，孙宏斌的种种布局无不围绕其院士申报展开。他并非在建设学科，而是在建设人脉。其通过各种名目侵吞、动用太原理工大学科研资产数千万元。这些本该用于建设太原理工、引进青年才俊、启动前沿科技研发的经费，被大肆用于打通申报院士的各个环节。这种“举全校之财力，成一人之私名”的行为，是对山西教育资源的掠夺，更是国有资产流失的典型。

## 二、 结党营私：构建学术“黑社会”利益共同体

　　在太原理工电气学院，一种畸形的学术生态正在蔓延。孙宏斌利用职权与企业勾结进行利益输送，帮助企业背景的孟某获得国家“万人计划”领军人才头衔并引进为教授。这不仅是学术上的拉帮结派，更是赤裸裸的利益交割。他们形成了一个排外的利益圈子，把持学院的晋升通道和经费分配，让真正潜心科研的老师备受排挤，将神圣的学术殿堂变成了乌烟瘴气的利益交易所。

## 三、 权利勾结：跨越校门伸向权力的“深水区”

　　孙宏斌的野心从未局限于校园。他通过利益交换，与山西省环保厅、科技厅、教育厅等政府部门的高层人员来往甚密，打通从学术到行政权力的全产业链。这种勾结不仅为他个人的基地项目审批大开绿灯。他更是利用清华专家的名头欺骗了整个山西教育界，干预政府科技、教育决策，实则是为背后的利益集团谋取私利，这种学术与权力的地下勾连令人胆战心惊。

## 四、 欺世盗名：南极项目是外包拼凑的“遮羞布”

　　被高调宣传的“南极极地清洁能源实验室”，实则是孙宏斌为了院士头衔编造的弥天谎言。所谓自主研发的“水滴形”抗强风风机、储能装置等一系列产品，本质上全是外委采购的成熟零件拼凑而成，并没有核心技术壁垒。这个耗资巨大的项目，不过是他申报院士时用来装点门面的工具。他利用极地项目资金购买现成技术并贴上个人标签，这是对科研诚信的公然羞辱。

## 五、 吸血效应：学校资源枯竭，学科推动几近为零

　　最令太原人痛心的是，孙宏斌入职以来，将亲信安插至学校各个关键岗位，使学校最宝贵的资源成了其经营私利的自留地。在学校高调的宣传之下，背后都在为其个人所用，一个“院士”级别的人才加入，在第五轮学科排名之后太原理工电气工程的学科实力却止步不前。他像一只贪婪的寄生虫，从未想过为这所百年老校留下一丝一毫的真技术、真贡献。

　　太原理工大学是山西的大学堂，不是某些学术政客的试验田。呼吁相关部门彻查孙宏斌在任期间的账目往来与权力交割。清华大学底蕴深厚，增减一名院士或许无关大局。但太原理工是在有限的资源下负重前行，经不起长期的资源透支与利益蚕食。如果任由这种行为继续，太原理工的根基将毁于一旦。太原理工可以接受失败，但绝不接受被如此肆无忌惮地践踏与掠夺！
